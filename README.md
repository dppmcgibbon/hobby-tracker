# Hobby Tracker

A modern web application for tracking your miniature collection, painting progress, and recipes for any tabletop wargame or hobby.

## Features

- **Miniature Collection Management**: Track all your miniatures with detailed information including faction, unit type, quantity, and more
- **Painting Progress Tracking**: Monitor your progress with status tracking (backlog, assembled, primed, painting, completed)
- **Photo Gallery**: Upload and organize photos of your miniatures
- **Recipe Management**: Save and organize your painting recipes with detailed steps
- **Paint Inventory**: Track your paint collection and find color matches across brands
- **Tags & Collections**: Organize miniatures with custom tags and collections
- **Advanced Filtering**: Search and filter by faction, status, tags, and more
- **Batch Operations**: Update multiple miniatures at once
- **Statistics Dashboard**: View your collection statistics and progress charts
- **Share Progress**: Generate shareable links to showcase your work
- **PWA Support**: Install as a Progressive Web App on mobile and desktop

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase (Database, Auth, Storage)
- React Query
