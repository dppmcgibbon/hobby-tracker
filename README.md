# Warhammer 40K Collection Tracker

A modern web application for tracking your Warhammer 40K miniature collection, painting progress, and recipes.

## Tech Stack

- Next.js 15 (App Router)
- TypeScript
- Tailwind CSS
- shadcn/ui
- Supabase (Database, Auth, Storage)
- React Query

## Getting Started

### Prerequisites

- Node.js 18.17+
- npm 9.0+
- Supabase account

### Installation

1. Clone the repository
2. Install dependencies:
